## To run locally

1. Clone the repository

```shell
git clone https://github.com/idolcs/idolcs.com.git
```

2. cd in the repo folder

```shell
cd idolcs.com
```

3. Install the packages

```shell
npm install
```

4. Start server

```shell
npm run dev
```
